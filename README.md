# Portafolio académico

¡Hola! Soy María Luz Mendieta de la Cruz 👋

Soy artista visual, educadora y mediadora cultural peruana. Me interesan las relaciones entre las artes visuales, la educación artística, la identidad, la memoria y los procesos creativos.

## Sobre este repositorio

Este repositorio fue creado como parte de mi proceso de aprendizaje en **Debug Colectiva**.

Su propósito es organizar y compartir progresivamente materiales propios relacionados con mi formación artística, educativa y académica.
